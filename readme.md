# Landing Page da Faculdade FGP Tatuí

Este é um projeto de landing page desenvolvido para a Faculdade FGP Tatuí, com o objetivo de oferecer informações sobre os cursos disponíveis. O projeto utiliza as seguintes tecnologias:

- [Vite](https://vitejs.dev/): Um construtor rápido para projetos web.
- [React](https://reactjs.org/): Uma biblioteca JavaScript para criar interfaces de usuário.
- [Styled Components](https://styled-components.com/): Uma biblioteca para estilização de componentes usando CSS-in-JS.
- [Vlibras](https://www.vlibras.gov.br/): Uma ferramenta de acessibilidade que traduz conteúdos digitais para Língua Brasileira de Sinais (Libras).
- [Userway](https://userway.org/): Uma solução de acessibilidade que permite aos usuários personalizar a experiência de navegação.

**Atenção: Este projeto encontra-se em fase de desenvolvimento.**

## Funcionalidades

- Apresentação dos cursos oferecidos pela Faculdade FGP Tatuí.
- Informações sobre o polo de apoio presencial.
- Acessibilidade aprimorada com recursos como Vlibras e Userway.

## Como Executar o Projeto

1. Clone este repositório em sua máquina local:

git clone https://github.com/seu-usuario/landing-page-fgp-tatui.git

2. Acesse o diretório do projeto:

cd landing-page-fgp-tatui

3. Instale as dependências do projeto:

npm install


4. Inicie o servidor de desenvolvimento:

npm run dev


5. Abra o seu navegador e acesse `http://localhost:3000` para visualizar a landing page.
